Telnet client

# How to install

## Debian / Ubuntu Linux

```bash
sudo apt-get update && sudo apt-get install telnet

```

## Fedora, CentOS, RedHat

```bash
sudo yum install telnet
```

## Mac Homebrew

```bash
brew install telnet
```
